Основные функции

  Программный продукт (далее ПП) создан для создания структуры организации.
  Возможности ПП:
    - созданиние и модификация подразделений;
    - добавление и редактирование сотрудников подразделений;
    - поиск сотрудников в подразделениях по определенным критериям;
    - экспорт/импорт данных в XML.

Краткое руководство

Подразделения (отделы организации) представлены в виде древовидной структуры.
Структура выглядит как раскрывающийся список: раскрыть подотделы можно кнопкой       закрыть кнопкой     
Отдел не имеющий подотделов не открывается, кнопка при этом меняется на пустую     

Добавить новый отдел можно нажав на        , при этом будет создан подотдел в этом отделе с указанным в строке ввода названием.

Отредактировать название можно нажав на

А удалить используя кнопку

Экспорт отдела можно сделать используя кнопку

Также над списком подразделений на этой странице находятся 2 кнопки:

           с помощью нее можно перейти на страницу поиска сотрудников в подразделениях


           эта кнопка предназначена для импорта данных из XML файла, созданного как при экспорте из данного ПП, так и из сторонних, имеющих
           нужную структуру данных


Для перехода к списку сотрудников отдела достаточно нажать на название отдела.

Страница подразделений

Страница сотрудников

Страница сотрудников представлена в виде таблицы, модификафия данных происходит с помощью кнопок с такими же иконками как и на странице подразделений:
- добавление сотрудника
- редактирование
- удаление

Список профессий в данной версии ПП изменить нельзя. Он задается при разработке ПП.
Поиск предназачен для выборки сотрудников по определенным критериям:
- можно выбрать всех сотрудников с определенной должностью;
- по определенному диапазону дат рождения;
- по имени или по определенному набору букв в имени сотрудников.

Поиск осуществляется с помощью кнопки 

Выборку можно сделать как по 1 вышеперечисленному критерию, так и по нескольким сразу.
Если не задать ни одного параметра будут выбраны все существующие сотрудники.
